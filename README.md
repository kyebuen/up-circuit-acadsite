# UP Circuit Academic Practice Website

An interactive practice platform for UP Circuit members to sharpen their EEE skills, inspired by Mr. Aren Cuevas' *EEEpen*.

## Story

> "Are there any sample exams (samplexes)?"

This is the number one question thrown around in EEE group chats during exam seasons. 

The academic drive exists, but one thing we've observed is that Mr. Aren Cuevas' site, EEEpen, is also shown to be highly effective to somewhat make reviewing interactive. Which is rare in studying engineering in general.

To further expand on this revelations, the **Academic Affairs Division** proposed the Academic Practice Website. A digital space to make reviewing more accessible and interactive.

## Objectives

* To be a giant repository of sample questions, for the members to practice. 
* Implement permanent, continously updating project such that it will be as reliable as the academic website.
* Provide an exclusive way for members to interactively practice different sets of questions. 

## Features

* A log-in system with the backend database is outsourced to a free "server" via Pythonanywhere.
* Ways to access the academic drive and to contribute to it.
* Featuring multiple EEE and non-EEE subjects
* A tiered question system, from beginners wanting to get the basics down, to practicing exam-level questions
