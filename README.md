# UP Circuit Academic Practice Website

An interactive practice platform for UP Circuit members to sharpen their EEE skills, inspired by Mr. Aren Cuevas' *EEEpen*.

## Story

> "Are there any sample exams (samplexes)?"

This is the number one question thrown around in EEE group chats during exam seasons. To solve this, the **Academic Affairs Division** proposed the Academic Practice Website. A digital space to make reviewing more accessible and interactive.

## Objectives

* To be a giant repository of sample questions, for the members to practice. 
* Implement permanent, continously updating project such that it will be as reliable as the academic website.
* Provide an exclusive way for members to interactively practice different sets of questions.
